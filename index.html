HTML Quiz


1. What does the HTML tag '<a>' represent?

Article This is its own tag, <article>, for housing standalone content.

Anchor Correct! '<a>' is short for this.

Audio This is its own tag, <audio>, used for playing audio files.

Aside This is its own tag, <aside>, used for displaying content as sidebars and such.


2. Which HTML element is used to define a table row?

<tr> Correct!

<td> This defines a table data cell.

<th> This defines a table header.

<table> This is what creates a table.


3. Which attribute is used to specify the URL of the page that the link goes to?

<href> Correct! This, paired with '<a>', determines a link's destination.

<src> This specifies the URL of an external recourse.

<link> This is typically used for linking to a style sheet.

<url> This isn't a tag attribute.
